## Installation

``` bash
# install dependencies
npm install

```

## Running the tests

``` bash
# run unit tests
npm run unit

# run all tests
npm test
```

## Build and Run

``` bash

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
